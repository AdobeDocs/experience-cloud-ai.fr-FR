# Data Engineering Agent

&lt;!— IL S&#39;AGIT D&#39;UN BROUILLON —>

Data Engineering Agent est un copilote optimisé par Agent Orchestrator et basé sur l’IA pour les équipes de données dans Adobe Experience Platform. L’agent est conçu pour gérer les tâches complexes tout au long du cycle de vie des données : modélisation des données, intégration, préparation des données SQL, gouvernance et gestion du cycle de vie. Avec le Data Engineering Agent, les ingénieurs et les architectes peuvent se déplacer plus rapidement avec une meilleure qualité des données et moins de travail manuel.

Data Engineering Agent est organisé en compétences dont vous pouvez tirer parti pour optimiser vos workflows.

| Compétences Data Engineering Agent | Description |
| --- | --- |
| Intégration des données | <strong>Intégration de données</strong> prend en charge les sources de lots telles que S3, Marketo et Data Landing Zone (DLZ). Ses fonctionnalités sont les suivantes : <ul><li>La connexion à diverses sources de données, notamment S3, Data Landing Zone, Marketo, etc.</li><li>Profilage des données sources (par exemple, valeurs distinctes, nulles, doublons et mesures de qualité de base).</li><li>Alignement des champs sources sur les groupes de champs XDM standard et les champs personnalisés.</li><li>Proposer et créer des flux de données d’ingestion de données (par exemple, déplacer des données de S3 ou Marketo vers des jeux de données RTCDP/CJA).</li></ul> |
| Qualité des données automatisée et enrichissement sémantique | <ul><li>Effectue des évaluations complètes de la qualité des données, notamment en identifiant les anomalies et en signalant les nombres d’enregistrements valides/non valides.</li><li>Recommande des améliorations de la qualité des données telles que la normalisation des types, le formatage des devises et la déduplication des identifiants.</li><li>Applique un enrichissement sémantique aux schémas en fonction de l’échantillonnage des données et de la reconnaissance intelligente des modèles de nommage.</li><li>Entraîne les utilisateurs à confirmer les enrichissements suggérés, en veillant à ce que les modifications de données finalisées et approuvées soient effectuées.</li></ul> |
| Data Distiller | |
| Collecte de données | |
| Validation des données | |

Avec Data Engineering Agent, vous pouvez vous assurer des avantages suivants :

- **Intégration plus rapide** : réduisez le temps nécessaire pour passer de plusieurs semaines de mappage manuel de schéma et de configuration de pipeline à plusieurs heures de configuration conversationnelle guidée.
- **Qualité de données supérieure** : moins d’enregistrements non valides et d’incidents de production dus à l’analyse de qualité des données et aux contrôles sémantiques intégrés.
- **Amélioration de la gouvernance et de la conformité** : les politiques de gouvernance sont jointes au moment de la conception (libellés, durée de vie, gestion des identités) plutôt que comme une réflexion après coup.
- **Des équipes de données plus productives** : le travail SQL/flux de données répétitif est automatisé afin que les ingénieurs se concentrent sur la conception et l’optimisation à plus forte valeur ajoutée.
- **En libre-service élargi** : les parties prenantes non expertes peuvent effectuer en toute sécurité des tâches de préparation de données courantes en libre-service grâce à des mécanismes de sécurisation.
